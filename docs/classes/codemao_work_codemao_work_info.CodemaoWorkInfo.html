Class CodemaoWorkInfo

编程猫作品信息类

  • 用于获取编程猫作品信息。
  • 所有属性均为Promise对象，当属性获取失败时访问该属性的值会被拒绝。

提供的作品信息详见类属性

具有以下特性：

  • 集成多个API接口，以确保在部分API接口信息获取失败时仍能提供尽可能完整的作品信息。
  • 内置懒加载和缓存机制，以减少不必要的请求。

集成API接口

已经集成的API接口

  • getWorkInfo
  • getWorkDetail
  • getNemoWorkPublicResource
  • getKittenWorkPublicResource

将来可能集成的API接口：

  • searchWorkByName

API优先级：

  • 优先使用 getWorkInfo 接口获取作品信息，该接口包含了作品的全部信息，但是容易出错。
  • 如果 getWorkInfo 接口获取失败，则使用 getWorkDetail 接口获取作品的大部分信息。
  • 如果 getWorkDetail 接口获取失败，则使用 getNemoWorkPublicResource 和 getKittenWorkPublicResource 接口获取作品的少部分信息。
  • 如果所有接口都获取失败，则抛出异常，对应属性的值会被拒绝。
